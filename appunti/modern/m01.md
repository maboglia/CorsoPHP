# 🛠 Modulo 1: Le Basi Moderne (Sintassi e Fondamenta)

L'obiettivo di questo blocco non è solo "scrivere codice", ma impostare un ambiente di lavoro professionale e capire perché PHP oggi è performante quanto (o più di) Node.js o Python.

## 1.1 L'Ambiente di Sviluppo: Addio XAMPP

Oggi nessuno sviluppatore senior installa PHP direttamente sul proprio Mac o PC. Usiamo la **virtualizzazione**.

* **Docker & DDEV/Lando:** Impareremo a creare un file di configurazione che "accende" un server Linux identico a quello di produzione con un solo comando.
* **PHP-FPM & Nginx:** Capire come il server comunica con l'interprete PHP (non più come modulo di Apache).
* **CLI vs Web:** Imparare a usare PHP da riga di comando per script e automazioni.

## 1.2 Type Safety (Il PHP "Serio")

PHP è nato come linguaggio debolmente tipizzato, ma oggi è diventato estremamente rigoroso.

* **Strict Types:** Usare `declare(strict_types=1);` in ogni file per evitare errori silenziosi.
* **Union Types & Mixed:** Gestire variabili che possono essere di più tipi (es. `int|float`).
* **Intersection Types:** Per interfacce complesse (es. `Iterator & Countable`).
* **Perché farlo?** Riduce i bug del 50% prima ancora di avviare il browser.

## 1.3 Lo zucchero sintattico di PHP 8.x

Scrivere meno codice per fare le stesse cose, meglio.

* **Match Expression:** Evoluzione dello `switch`. Restituisce un valore, è più corto e non permette dimenticanze (errori se manca un caso).
```php
$status = match($code) {
    200 => 'Successo',
    404 => 'Non trovato',
    default => 'Errore ignoto',
};

```


* **Nullsafe Operator:** Evitare catene di `if ($obj !== null)`. Usiamo `$obj?->metodo()`.
* **Named Arguments:** Passare parametri alle funzioni specificando il nome, così l'ordine non conta più e il codice è leggibile come un libro.

## 1.4 Stringhe e Array: Le nuove funzioni

* **Stringhe:** `str_contains()`, `str_starts_with()`, `str_ends_with()` (finalmente funzioni native leggibili).
* **Array Destructuring:** Estrarre valori dagli array in variabili al volo con la sintassi `[]`.

**Il "System Health Checker":**
Creeremo un piccolo script da riga di comando che:

1. Accetta un elenco di URL.
2. Controlla se sono online usando le nuove funzioni di stringa e i tipi rigorosi.
