# 1.4 Stringhe e Array: le nuove funzioni (PHP 8.x)

Obiettivo: usare le nuove funzioni native di PHP per scrivere codice **più leggibile**, evitando `strpos()` e controlli complicati, e imparare a “spacchettare” array in modo elegante.

# 1) Stringhe: finalmente funzioni chiare

Per anni in PHP si usava `strpos()` per verificare se una stringa contiene un testo, ma era scomodo e pieno di trappole (`0` è falsy!).

PHP 8 introduce funzioni native molto più leggibili.

---

## 1.1 `str_contains()`

Verifica se una stringa contiene un’altra stringa.

```php
<?php
declare(strict_types=1);

$testo = "Benvenuto nel corso PHP";

if (str_contains($testo, "corso")) {
    echo "Trovato!";
}
```

✅ ritorna `true/false`
✅ leggibile
✅ niente bug da `strpos()`

---

## 1.2 `str_starts_with()`

Controlla se una stringa inizia con un prefisso.

```php
$url = "https://example.com";

if (str_starts_with($url, "https://")) {
    echo "Connessione sicura";
}
```

Caso tipico: validazione URL, prefissi file, prefissi codice.

---

## 1.3 `str_ends_with()`

Controlla se una stringa finisce con un suffisso.

```php
$file = "documento.pdf";

if (str_ends_with($file, ".pdf")) {
    echo "È un PDF";
}
```

Caso tipico: estensioni file, validazione input, controlli email.

---

## 1.4 Confronto con il vecchio metodo (`strpos`)

Metodo vecchio:

```php
if (strpos($testo, "corso") !== false) {
    echo "Trovato";
}
```

⚠️ facile sbagliare con:

```php
if (strpos($testo, "Ben") == false) {
    // ERRORE: "Ben" è in posizione 0, ma 0 == false
}
```

👉 con `str_contains()` questo problema sparisce.

# 2) Array Destructuring (spacchettare array)

## Concetto

Puoi estrarre valori da un array direttamente in variabili:

```php
$persona = ["Mario", "Rossi", 25];

[$nome, $cognome, $eta] = $persona;

echo $nome;     // Mario
echo $cognome;  // Rossi
echo $eta;      // 25
```

Molto più pulito di:

```php
$nome = $persona[0];
$cognome = $persona[1];
$eta = $persona[2];
```

---

## 2.1 Saltare elementi

```php
$dati = ["Mario", "Rossi", 25, "Roma"];

[$nome, , $eta] = $dati;

echo $nome; // Mario
echo $eta;  // 25
```

La virgola vuota indica “salta quel valore”.

---

## 2.2 Destructuring con array associativi

Con gli array associativi puoi estrarre usando le chiavi:

```php
$utente = [
    "id" => 10,
    "nome" => "Anna",
    "email" => "anna@test.it"
];

["id" => $id, "nome" => $nome] = $utente;

echo $id;
echo $nome;
```

---

## 2.3 Caso pratico: funzione che restituisce più valori

```php
function calcolaStatistiche(array $numeri): array {
    return [
        "min" => min($numeri),
        "max" => max($numeri),
        "media" => array_sum($numeri) / count($numeri)
    ];
}

$valori = [10, 20, 30];

["min" => $min, "max" => $max, "media" => $media] = calcolaStatistiche($valori);

echo $min;
echo $max;
echo $media;
```

👉 codice elegante, tipico in PHP moderno.

# 3) Mini esercitazione da laboratorio

## Obiettivo

Scrivere uno script che:

* controlla se una mail è valida (contiene `@`)
* controlla se finisce con `.it`
* spacchetta nome e cognome da un array

---

### Soluzione esempio

```php
<?php
declare(strict_types=1);

$email = "mario.rossi@test.it";

if (!str_contains($email, "@")) {
    die("Email non valida\n");
}

if (!str_ends_with($email, ".it")) {
    die("Email non italiana\n");
}

$persona = ["Mario", "Rossi"];
[$nome, $cognome] = $persona;

echo "Email OK: $email\n";
echo "Utente: $nome $cognome\n";
```
